# BACnet Commissioning Operator (Tauri)

Minimal desktop shell around `tools/runtime/app.py`.

See **[`docs/packaging/tauri-operator-desktop.md`](../../docs/packaging/tauri-operator-desktop.md)** for prerequisites, build, and environment variables.

Quick build:

```bash
npm ci
npx tauri build
```

Development:

```bash
npm run tauri dev
```
